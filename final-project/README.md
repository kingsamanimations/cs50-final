### Description:
King Sam Media Portfolio is a professional, multi-page portfolio website showcasing my work as a digital creator, animator, and music producer. The website features:

- **Interactive Video Galleries**: Custom-built slideshows for Blender tutorials, Scratch animations, and YouTube Shorts
- **SoundCloud Integration**: Embedded music player featuring my EDM productions
- **Responsive Design**: Fully mobile-friendly with Bootstrap 5
- **Dark Mode**: User preference saved with localStorage
- **Dynamic Video Backgrounds**: Engaging hero sections on each page
- **Professional Navigation**: Smooth scrolling and active page indicators

### Technologies Used:
- HTML5
- CSS3 (Custom animations, gradients, and responsive design)
- JavaScript (Slideshow functionality, theme toggle, localStorage)
- Bootstrap 5
- Font Awesome Icons
- YouTube & SoundCloud Embed APIs

### Features:
1. **Video Slideshow System**: Built from scratch with JavaScript
2. **Three Separate Portfolio Pages**: Each with unique content categories
3. **SoundCloud Integration**: Direct music streaming
4. **Theme Persistence**: Dark mode saves user preference
5. **Professional Design**: Modern gradients, animations, and hover effects

### Design Decisions:
- Used single-page sections for easy navigation
- Implemented custom slideshow instead of carousel plugin for more control
- Yellow frames for video containers to create visual consistency
- Dark mode as an optional feature for better accessibility
